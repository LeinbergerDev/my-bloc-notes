HTML5 Document Structure

HyperText Markup Language

All HTML documents should have the same basic sructure. They should all start with a <!DOCTYPE html> tag that identifies that this document is fomated in HTML5. Next comes the <html> tag. This is the root tag of the document and all other tags should within it. Next there is the <head> tag. This is where the <title>, <meta>, <script> and <link> will be found. This area is not actually displayed by the browser. The <title> controls what the browser displays in the browser tab. The <meta> allows the user to create meta tags for the site. See meta tag list. The <link> tag allows us to add a link to a document. This is commonly used to add a CSS styles sheet to the document. See Link Tag for details. The <script> is used to import script files into the document. See script tag for more details.

Meta tags

<meta charset="UTF-8">
Sets the character set for the document.
<meta name="description" content="Notes on HTML5 from bloc training course and my own suplimental research.">
Adds a description for the content of the document. Search engines use this tag to find out what the page is about.
<meta name="keywords" content="HTML5">
Adds keywords to tell the search engine what the content of the page is about.
<meta name="author" content="Jason Leinberger">
Name of the person who created the document.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Sets the initial size of the viewport

Link Tag Syntax

Script Tag Syntax
